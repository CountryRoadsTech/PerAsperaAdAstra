# Per Aspera Ad Astra - Through Hardship to the Stars!

A Ruby on Rails application to calculate, visualize, and render videos of orbital mechanics.

 * Ruby 3+

 * Ruby on Rails 7+

 * Hotwire

 * StimulusJS

 * ThreeJS

 * D3.JS

 * FFmpeg

 * SciRuby NMatrix

 * GNU Scientific Library
